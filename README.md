🚖 Uber Trip Analysis Dashboard (Power BI)
📌 Project Overview

This project presents an interactive Uber Trip Analysis Dashboard developed using Microsoft Power BI.

The dashboard provides valuable business insights into Uber ride bookings, revenue generation, trip distance, booking trends, vehicle performance, payment methods, and customer behavior.

🎯 Objective

The objective of this project is to analyze Uber trip data and help business stakeholders understand:

Total Bookings
Total Revenue
Average Trip Distance
Average Trip Time
Booking Trends
Vehicle Type Performance
Payment Analysis
Location Analysis
